第十二章 ICA 独立成分分析

约 3132 字大约 10 分钟

2024-11-08

在这一章节中，我们将学习 独立成分分析（Independent Components Analysis）。与主成分分析（PCA）相似，这将为我们提供一种新的数据表示方式。然而，ICA 的目标却与 PCA 有很大不同。

下面，我们介绍一个称为"鸡尾酒会问题"的例子。在这个例子中，有多个说话者同时在派对上讲话，并且一个麦克风记录下来的一定只是这些说话者声音的叠加混合。但假设我们有nnn个不同位置的麦克风，并且由于每个麦克风到各个说话者的距离不同，每个麦克风记录下的声音都是这些说话者声音的不同组合。使用这些麦克风录音，我们能否从中分离出每个说话者的语音信号呢？

为了形式化这一问题，我们设想存在某些数据s∈Rns \in \mathbb{R}^ns∈Rn，它是由nnn个独立源生成的。我们观察到的数据是：

其中AAA是一个未知的方阵，称为混合矩阵（mixing matrix）。通过反复地观测，我们得到了一个数据集{x(i):i=1,…,m}\{x^{(i)}: i = 1, \dots, m\}{x(i):i=1,…,m}，我们的目标是恢复这些源信号s(i)s^{(i)}s(i)，即生成我们数据的源信号 (x(i)=As(i)x^{(i)} = As^{(i)}x(i)=As(i))。

ICA 的模糊性

具体来说，设PPP为任意n×nn \times nn×n的置换矩阵。这意味着PPP的每一行和每一列都有且仅有一个"1"。下面是一些置换矩阵的例子：

也就是说，没有办法恢复wiw_iwi​的正确缩放。举例来说，如果AAA被替换为2A2A2A，并且s(i)s^{(i)}s(i)被替换为0.5s(i)0.5s^{(i)}0.5s(i)，那么我们观察到的x(i)=2A⋅(0.5)s(i)x^{(i)} = 2A \cdot (0.5)s^{(i)}x(i)=2A⋅(0.5)s(i)将保持不变。广义上讲，如果AAA的某一列被按因子α\alphaα缩放，而相应的源信号则按1/α1/\alpha1/α缩放，那么单从x(i)x^{(i)}x(i)来看，仍然无法判断出这种变化。因此，我们无法恢复出源信号的"正确"缩放。然而，对于我们关心的应用——包括鸡尾酒会问题——这种不确定性同样也不重要。具体来说，按某个正的因子缩放说话者的语音信号sj(i)s_j^{(i)}sj(i)​只会影响该说话者语音的音量。同样，符号的变化也不重要，因为sj(i)s_j^{(i)}sj(i)​和−sj(i)-s_j^{(i)}−sj(i)​在播放时听起来是一样的，都在扬声器中是同样的音量大小。因此，如果通过算法找到的wiw_iwi​被任意非零实数缩放，相应恢复出来的源信号si=wiTxs_i = w_i^T xsi​=wiT​x也会按同样的因子缩放；但这通常并不重要。

上面这些是 ICA 唯一模糊性来源吗？事实证明，只要源信号sis_isi​是非高斯分布的，就没有其他的不确定性因素。为了说明高斯分布下数据中的困难情况，考虑一个例子，其中n=2n = 2n=2，并且s∼N(0,I)s \sim \mathcal{N}(0, I)s∼N(0,I)，其中III是2×22 \times 22×2单位矩阵。注意标准正态分布N(0,I)\mathcal{N}(0, I)N(0,I)的密度轮廓是以原点为中心的圆，并且密度是旋转对称的。

现在，假设我们观察到x=Asx = Asx=As，其中AAA是我们的混合矩阵。xxx的分布也将是高斯分布，均值为零且协方差E[xxT]=E[AssTAT]=AATE[xx^T] = E[Ass^T A^T] = AA^TE[xxT]=E[AssTAT]=AAT。现在，设RRR为任意正交矩阵（形式上是旋转/反射矩阵），使得RRT=RTR=IRR^T = R^T R = IRRT=RTR=I，并设A′=ARA' = ARA′=AR。如果数据是通过AAA混合的，而不是通过A′A'A′混合的，我们将观察到x′=A′sx' = A'sx′=A′s。x′x'x′的分布仍然是高斯分布，均值为零且协方差E[x′(x′)T]=E[AssT(A′)T]=E[ARssT(AR)T]=ARRTAT=AATE[x'(x')^T] = E[Ass^T(A')^T] = E[ARss^T(AR)^T] = ARR^TA^T= AA^TE[x′(x′)T]=E[AssT(A′)T]=E[ARssT(AR)T]=ARRTAT=AAT。因此，我们将从N(0,AAT)\mathcal{N}(0, AA^T)N(0,AAT)分布中观察数据。因此，无法从中分辨出数据是使用AAA还是A′A'A′混合的。换句话说，只要混合矩阵中存在一个任意的旋转分量，无法通过数据推导出来，我们就无法恢复原始源信号。

我们的论点是基于多元标准正态分布是旋转对称的这个定理。这些情况使得 ICA 面对高斯分布的数据的时候很无力，但是只要数据不是高斯分布的，然后再有充足的数据，那就还是能恢复出 n 个独立的声源的。

ICA 算法

我们现在准备推导 ICA 算法。我们描述的算法源自于 Bell 和 Sejnowski，他们对该算法的解释是将其视为一种最大似然估计。（这与他们最初的解释不同，最初的解释涉及到一个叫做信息最大化原理的复杂概念，这在现代 ICA 理解中已经不再需要。）
